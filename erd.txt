============================================================
   ERD - Metro Interstate Traffic Volume Database
============================================================

  +-------------------+         +-------------------------+
  |     holidays      |         |   weather_conditions    |
  +-------------------+         +-------------------------+
  | PK holiday_id INT |         | PK weather_id INT       |
  |    holiday_name   |         |    weather_main         |
  |    VARCHAR(100)   |         |    VARCHAR(50)          |
  +-------------------+         |    weather_description  |
          |                     |    VARCHAR(100)         |
          | 1                   +-------------------------+
          |                               |
          | (nullable)                    | 1
          |                               |
          | *                             | *
  +-----------------------------------------------+
  |              traffic_records                   |
  +-----------------------------------------------+
  | PK record_id      INT AUTO_INCREMENT           |
  |    date_time       DATETIME          NOT NULL   |
  |    temp            DECIMAL(6,2)                 |
  |    rain_1h         DECIMAL(8,2)                 |
  |    snow_1h         DECIMAL(8,2)                 |
  |    clouds_all      INT                          |
  |    traffic_volume  INT                          |
  | FK holiday_id      INT  -> holidays(holiday_id) |
  | FK weather_id      INT  -> weather_conditions   |
  +-----------------------------------------------+

============================================================
   Relationships
============================================================

  holidays (1) ----< (many) traffic_records
      One holiday can appear in many traffic records.
      holiday_id is NULLABLE (most records have no holiday).

  weather_conditions (1) ----< (many) traffic_records
      One weather condition can appear in many traffic records.
      weather_id is NOT NULL (every record has a weather condition).

============================================================
   Cardinality Summary
============================================================

  holidays              1 : N   traffic_records
  weather_conditions    1 : N   traffic_records
